# String.prototype.replace

相关概念参考mdn:replace

这里主要讲述replace方法的第二个参数为函数，一般来说，replace方法在使用时第二个参数为字符串，但第二个参数为函数时能够给字符串的处理带来极大的方便

# 示例

将-拼接的变量变为驼峰命名的变量，如hello-world转化为helloWorld.

一种实现思路是找到所有的-对应的索引，然后将每个索引的后一位对应的字符大写，显然这种实现方式不够简洁，没有体现正则表达式以及replace方法的强大之处，那么如何使用replace方法实现 驼峰转化呢。

# 方法

const re = /-(\w)/g
const camelize = str => str.replace(re, (_, c) => c ? c.toUpperCase() : '')
1
2

这里replace方法的第二个参数为函数，借助于正则表达式re实现了camelize方法，从而将变量名驼峰化

# 总结

当使用函数作为第二个参数时，第一个参数一般都为正则，这要求对正则表达式有一定的熟悉，然后借助于replace方法能够对字符串进行操作，得到想要的字符串。
